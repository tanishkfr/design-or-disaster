# Design or Disaster

**Design criticism as incompatible perception.**

Design or Disaster is a research-through-design investigation in which the visitor files a spatial evidence plate before seeing how five fallible jurors read the same interface.

Every plate shares one coordinate system. Changing lenses changes what counts as evidence; the visitor’s marks remain beneath the comparison. The final sealed case opens no outside plate.

## Interaction

**Look → choose a lens → mark evidence → rule → seal → compare plates.**

## Run

```sh
npm install
npm run dev
```

Verification:

```sh
npm run lint
npm run build
```
